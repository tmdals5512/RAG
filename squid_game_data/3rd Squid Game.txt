Title: 3rd Squid Game
URL: https://squid-game.fandom.com/wiki/3rd_Squid_Game
--------------------------------------------------
The 3rd Squid Game was the third annual Squid Game competition that took place in 1990. Nothing is known about the set of games, except that a man named Park Pil-sam won in the end. Player files for the game appear briefly in "Gganbu".

Known players[]
Player 028 - Park Pil-sam (winner)
Player 057
Player 058 - Son Jeong-hee
